# JSTL 進階迭代與狀態處理 (Day 2-2)

本單元設計為 120 分鐘的實戰課程，深入講解 `<c:forEach>` 的進階用法以及 `<c:forTokens>` 字串解析。

## 1. 課程大綱
- **[30m] forEach 基礎**：遍歷 List、Map、陣列。
- **[40m] 迭代控制**：`begin`、`end`、`step` 與 `varStatus` 屬性。
- **[30m] 字串迭代**：`<c:forTokens>` 切割 CSV 等分隔字串。
- **[20m] 綜合實戰**：動態產生斑馬紋表格與條件式樣式。

## 2. 核心觀念：迭代器的強大能力
1. **varStatus 物件**：提供 `index`（從 0 開始）、`count`（從 1 開始）、`first`、`last`，便於實作奇偶列、首尾特殊樣式。
2. **子集合控制**：不需要修改原始集合，即可透過 `begin`/`end` 擷取部分資料。
3. **步長 (step)**：可跳躍式迭代，適用於分頁或每隔 N 項取資料。

## 3. 實作任務
### 實驗 1：間隔取樣
- 給定一個分數清單 `[90,85,77,92,60,45,88]`。
- 使用 `step="2"` 並搭配 `begin="0" end="6"`，只顯示索引為偶數的成績。

### 實驗 2：CSV 轉按鈕組
- 字串 `"壽險,醫療,意外,儲蓄,投資,防疫"`。
- 使用 `<c:forTokens>` 切割並將每個標籤顯示為 `<button>`。

### 實驗 3：動態表格與狀態列
- 遍歷保單清單（含 `active` 布林欄位）。
- 若 `active` 為 `false`，該行文字加上刪除線；價格大於 10000 顯示「高階」圖示。
- 利用 `varStatus.count` 實作斑馬紋（偶數行灰底）。

## 4. 關鍵語法清單
- `<c:forEach items="${list}" var="item" varStatus="st">`
- `<c:forEach begin="1" end="10" step="2" var="i">`
- `<c:forTokens items="${csv}" delims="," var="token">`

## 5. 課後練習 (30m)
### 挑戰 1：動態產生月份下拉選單
- **目標**：使用 `begin`/`end` 產生 1 到 12 的 `<option>`。
- **實作**：在 JSP 中建立一個月份選擇器，並將目前月份（例如三月）設定為 `selected`。

### 挑戰 2：奇偶數著色器
- **目標**：練習 `varStatus` 的 `even`/`odd` 判斷。
- **實作**：展示 1~20 的數字方塊，偶數背景淺藍色，奇數背景淺灰色。
