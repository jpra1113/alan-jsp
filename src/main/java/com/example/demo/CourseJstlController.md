# JSTL & EL 深度實踐教學大綱 (Day 2)

## 課程目標
1. 掌握 JSTL 核心標籤庫 (`c:`) 的邏輯控制與變數管理。
2. 學習如何使用 EL (Expression Language) 簡化資料存取。
3. 實作進階迭代與動態 URL 生成。

## 上午：邏輯與異常管理
- **變數管理**：使用 `<c:set>` 在不同 Scope (Page, Session) 暫存資料。
- **條件控制**：
    - `<c:if>`：簡單的 true/false 判斷。
    - `<c:choose>` / `<c:when>` / `<c:otherwise>`：處理多重分支邏輯（如權限判斷）。
- **異常捕獲**：使用 `<c:catch>` 避免頁面因為小錯誤而完全崩潰。

## 下午：迭代、解析與導向
- **進階迭代**：
    - `<c:forEach>`：利用 `begin`, `end`, `step` 處理分頁或區段資料。
    - `varStatus` 屬性：獲取當前索引、計數、是否為首項/末項。
- **字串解析**：使用 `<c:forTokens>` 處理 CSV 或逗號分隔的標籤資料。
- **URL 處理**：
    - `<c:url>`：自動處理 Context Path。
    - `<c:param>`：對參數進行自動 URL 編碼，防止亂碼。
